# IMPORTANT NOTES:
#
# 1. Generate APP_KEY:
#    Run this in Dokploy terminal: php artisan key:generate --show
#    Copy the output (starts with "base64:")
#
# 2. Database credentials:
#    - DB_HOST should match your Dokploy database name
#    - DB_PASSWORD is shown when you create the database
#
# 3. Matrix passwords:
#    - Generate with: openssl rand -hex 32
#    - Or use the deployment/create-matrix-bot.sh script
#
# 4. Laravel API Token:
#    Generate in tinker:
#    php artisan tinker
#    > $user = User::first();
#    > $token = $user->createToken('minerva-bot')->plainTextToken;
#    > echo $token;
#
# 5. Minerva AI:
#    - Ollama is free and runs locally (default)
#    - Or use Claude/OpenAI with API key
